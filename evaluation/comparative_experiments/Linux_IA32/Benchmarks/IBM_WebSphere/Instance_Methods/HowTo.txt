Build class files from source code:
===================================

$: javac -Xrealtime NHRT1000Methods.java

Create JAR-archive:
===================

$: jar -cvf NHRT1000Methods.jar *.class

Build JNI header for timer library:
===================================

$: javah NHRT1000Methods

Build timer library:
====================

- Edit the file 'timer.c', so that it includes the header
file 'NHRT1000Methods.h'. Ensure that the function name matches
that in the header file.

- Edit the Makefile so that it includes the correct path to
the installation of IBM WebSphere and run the Makefile.

Build the shared cache (AOT compilation):
=========================================

$: admincache -Xrealtime -populate -noisy -aot /your/path/Instance_Methods/NHRT1000Methods.jar -cacheName NHRT1000MethodsCache -classpath /your/path/Instance_Methods/NHRT1000Methods.jar -printvmargs

Run benchmark:
==============

- Ensure that the environment variable PATH includes IBM WebSphere, e.g., by issuing

$: export LD_LIBRARY_PATH=.
$: export PATH=/path/to/WebSphere/ibm/ibm-wrt-i386-60/jre/bin:/ath/to/WebSphere/ibm/ibm-wrt-i386-60/bin:$PATH 

- JIT mode:

$: chrt 79 java -Xrealtime -classpath /your/path/Instance_Methods/NHRT1000Methods.jar NHRT1000Methods $(date +%s::%N)

- AOT mode:

chrt 79 java -Xrealtime -Xshareclasses:name=NHRT1000MethodsCache -classpath /your/path/Instance_Methods/NHRT1000Methods.jar -Xaot NHRT1000Methods $(date +%s::%N)
