Build class files from source code:
===================================

$: jamaicac -useTarget targetlinux-x86 -d /your/path/Static_Class_Methods/classes/ NHRT1000ClassStaticMethods.java


Build JNI header for timer library:
===================================

$: cd /your/path/Static_Class_Methods/classes
$: jamaicah NHRT1000ClassStaticMethods

Build timer library:
====================

- Edit the file 'timer.c', so that it includes the header
file 'NHRT1000ClassStaticMethods.h'. Ensure that the function name matches
that in the header file.

- Run the Makefile.

Build the executable (AOT compilation):
=======================================

$: jamaicabuilder -target=targetlinux-x86 -cp=/your/path/Static_Class_Methods/classes/ -lazy=false -compile NHRT1000ClassStaticMethods

Run benchmark:
==============

$: export JAMAICAVM_IMMORTALSIZE=4M JAMAICAVM_LAZY=false JAMAICA_SCHEDULING=OTHER

- JIT mode:

$: chrt 79 jamaicavm -cp /your/path/Static_Class_Methods/classes/ NHRT1000ClassStaticMethods $(date +%s::%N)

- AOT mode:

$: chrt 79 ./NHRT1000ClassStaticMethods $(date +%s::%N)
